# Fleet Management Platform

Plateforme de gestion de flotte de véhicules - Spring Boot + Angular

## Technologies
- Backend : Spring Boot 3.2 + MySQL 8
- Frontend : Angular 21 + Nginx
- Docker + Docker Compose

## Lancer avec Docker
```bash
docker-compose up --build -d
# Frontend : http://localhost:4200
# Backend  : http://localhost:9090/api
```

## Entités
- Vehicule, Chauffeur, TrajetMission, Consommation
